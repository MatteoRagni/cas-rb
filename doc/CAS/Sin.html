    diff(v) click to toggle source

    Return the derivative of the `sin(x)` function using the chain rule. The input is a `CAS::Op` because it can handle derivatives with respect to functions.

    “`

    d

    – sin(f(x)) = f'(x) cos(fx) dx “`

    <- `CAS::Op` object of the derivative -> `CAS::Op` a derivated object, or `CAS::Zero` for constants

    # File lib/functions/fnc-trig.rb, line 25
def diff(v)
  if @x.depend? v
    return @x.diff(v) * CAS.cos(@x)
  else
    return CAS::Zero
  end
end
