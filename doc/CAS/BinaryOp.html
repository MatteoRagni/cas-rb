    new(x, y) click to toggle source

    The binary operator inherits from the `CAS::Op`, even if it is defined as a node with two possible branches. This is particular of the basic operations. The two basic nodes shares the *same* interface, so all the operations do not need to know which kind of node they are handling.

    <- `CAS::Op` left argument of the node or `Numeric` to be converted in `CAS::Constant` <- `CAS::Op` right argument of the node or `Numeric` to be converted in `CAS::Constant` -> `CAS::BinaryOp` instance

    # File lib/op.rb, line 289
def initialize(x, y)
  if x.is_a? Numeric
    case x
    when 0
      x = CAS::Zero
    when 1
      x = CAS::One
    else
      x = CAS.const x
    end
  end
  if y.is_a? Numeric
    case y
    when 0
      y = CAS::Zero
    when 1
      y = CAS::One
    else
      y = CAS.const y
    end
  end
  CAS::Help.assert(x, CAS::Op)
  CAS::Help.assert(y, CAS::Op)

  @x = x
  @y = y
end

def ==(op)
  CAS::Help.assert(op, CAS::Op)
  if op.is_a? CAS::BinaryOp
    return (self.class == op.class and @x == op.x and @y == op.y)
  else
    return false
  end
end

    diff(v) click to toggle source

    This method returns an array with the derivatives of the two branches of the node. This method is usually called by child classes, and it is not intended to be used directly.

    <- `CAS::Op` operation to differentiate against -> `Array` of differentiated branches ([0] for left, [1] for right)

    # File lib/op.rb, line 330
def diff(v)
  CAS::Help.assert(v, CAS::Op)
  left, right = CAS::Zero, CAS::Zero

  left = @x.diff(v) if @x.depend? v
  right = @y.diff(v) if @y.depend? v

  return left, right
end

def subs(dt)
  CAS::Help.assert(dt, Hash)
  if dt.keys.include? @x
    if dt[@x].is_a? CAS::Op
      @x = dt[@x]
    elsif dt[@x].is_a? Numeric
      @x = CAS::const dt[@x]
    else
      raise CASError, "Impossible subs. Received a #{dt[@x].class} = #{dt[@x]}"
    end
  else
    @x.subs(dt)
  end
  if dt.keys.include? @y
    if dt[@y].is_a? CAS::Op
      @y = dt[@y]
    elsif dt[@y].is_a? Numeric
      @y = CAS::const dt[@y]
    else
      raise CASError, "Impossible subs. Received a #{dt[@y].class} = #{dt[@y]}"
    end
  else
    @y.subs(dt)
  end
  return self
end
